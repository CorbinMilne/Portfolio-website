Home Security Monitoring System

I built a home automation and security system using a Raspberry Pi 5, written in C. The system uses a PIR motion sensor to detect movement, triggering a camera module to capture images in real time. Events are timestamped and logged locally, while a notification system sends alert emails with photo attachments using curl and SMTP scripting. To improve usability, I incorporated button input for arming/disarming the system and LED indicators for status feedback. This project helped me apply low-level hardware interfacing, real-time event handling, and scripting for automation in a real-world setting.

Space Invaders

I built a fully functional clone of the classic Space Invaders arcade game in Java, applying object-oriented design principles such as encapsulation and polymorphism. The game includes enemy waves, player-controlled shooting, collision detection, and score tracking, all managed through event-driven logic. I structured the program using modular classes for the player, enemy, bullets, and game engine, giving me hands-on experience with GUI rendering, keyboard input handling, and game loop timing.
